# OceanOS

**OceanOS is the desktop runtime for OceanOS Agent.**

Users get one guided install and one polished desktop: config, chat, agents, skills, schedules, models, profiles, memory, and tools. Think of it as the operating layer for OceanOS — not just an Electron wrapper, but the owner of setup, state, and ecosystem access.

## Mission Control

Every OceanOS capability becomes a first-class surface inside the desktop:

- Install, update, and configure OceanOS without touching the CLI directly.
- Chat, automation, and agent runtime in one workspace.
- Skills/registry/marketplace surface to install reusable capabilities.
- Agents/runtime controls to run tasks, schedules, and tools.
- Memory, profiles, and settings centralized instead of scattered across config files.
- Model selection and provider management built in.

The long-term goal is a public marketplace/runtime where users install OceanOS skills, publish agents, and run AI apps from the same desktop/registry layer.
